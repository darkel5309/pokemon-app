DESCRIPCIÓN

Este proyecto es una aplicación cliente sobre la Pokédex de Pokémon utilizando la tecnología de desarrollo front-end Angular y consumiendo la API pública y gratuita que ofrece https://pokeapi.co/. En ella, mostramos todos los Pokémon existentes actualmente, ofrecemos un buscador y mostramos toda la información a cerca de cada uno de ellos.
